셀룰러 네트워크와 셀 구조 개요

셀룰러 네트워크는 서비스 지역을 작은 셀로 나누어 주파수 재사용과 용량 극대화를 도모합니다. 기지국이 각 셀을 담당하며, 육각형 모델을 기준으로 주파수 재사용 패턴을 설계합니다. 사용자가 이동하면 핸드오프를 통해 통화나 데이터가 끊김 없이 유지됩니다.

셀룰러 네트워크 구성 요소

기지국(BTS)은 단말과 무선 통신을, BSC는 기지국 관리를, MSC는 통화 스위칭과 외부망 연결을 담당합니다. HLR/VLR DB를 통해 가입자 정보와 현재 위치를 관리하며, MSC가 이를 기반으로 최적 경로를 찾아 연결합니다.

이동통신 세대별 발전

1G는 아날로그 음성, 2G는 디지털화와 SMS, 3G는 멀티미디어 데이터, 4G는 LTE 기반 광대역 인터넷으로 발전했습니다. 각 세대는 데이터 속도와 서비스 범위를 확대하며 모바일 환경을 혁신시켰습니다.

무선 자원 할당 방식

1G는 FDMA, 2G는 FDMA+TDMA(GSM)와 CDMA(IS-95), 3G는 WCDMA, 4G는 OFDMA를 활용했습니다. 이들은 주파수, 시간, 코드, 부반송파를 다중 접속과 효율적 자원 활용에 맞춰 결합한 방식입니다.

요약 및 학습 팁

요약: 셀룰러 망은 셀 구조와 주파수 재사용, 계층적 네트워크(기지국-BSC-MSC)로 구성되며, 세대별 기술은 데이터 중심으로 발전해왔습니다.

학습 팁: 세대별 핵심 키워드(예: 1G-아날로그, 4G-LTE)를 정리하고, 기술 연표와 주요 구성 요소-역할 매칭표를 직접 작성해보세요. 이동 중 체험한 통화, 데이터 품질을 개념과 연결지으면 이해가 깊어집니다.
